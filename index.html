Autonomous Firefighting Robot

Published IEEE Xplore | Presented at ICEAMST 2025

Problem Statement

Traditional firefighting systems are expensive and require human intervention. Small residential fires need a quick, autonomous response to prevent escalation.

Solution

Designed a compact mobile robot that navigates confined spaces, detects fire via thermal/IR sensors, and activates a suppression mechanism automatically.

Key Outcomes

  • Successfully built working prototype.
  • Reduced response time in simulated tests.
  • Research paper published in IEEE.

Smart Garden Automated System

Overview

An IoT-based automation project designed to monitor environmental conditions and automate irrigation for plants to reduce water wastage.

Hardware Used

  • ESP8266 Microcontroller
  • Soil Moisture Sensor & DHT11
  • Water Pumps

Features

Real-time dashboard via Blynk, auto-shutoff for water conservation, and LCD status display.

LoRa-Based Remote Agriculture

The Challenge

Remote agricultural fields often lack Wi-Fi or cellular coverage, making standard IoT monitoring impossible.

The Innovation

Used LoRa (Long Range) communication technology to transmit sensor data over kilometers without needing the internet. The system involves a Transmitter Node in the field and a Receiver Node at the base station.

Skills Demonstrated

  • Wireless Communication Protocols
  • Low-power embedded design
  • System Architecture
