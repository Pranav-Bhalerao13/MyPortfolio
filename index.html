LoRa-Based Remote Agriculture

The Challenge

Remote agricultural fields often lack Wi-Fi or cellular coverage, making standard IoT monitoring impossible for deep-field management.

The Innovation

Used LoRa (Long Range) communication technology to transmit sensor data over kilometers without needing the internet. The system involves a Transmitter Node in the field and a Receiver Node at the base station.

Skills Demonstrated

  • Wireless Communication Protocols (LoRaWAN)
  • Low-power embedded design
  • System Architecture and Data Handling

Autonomous Firefighting Robot

Published IEEE Xplore | Presented at ICEAMST 2025

Problem Statement

Traditional firefighting systems are expensive and require human intervention. Small residential fires need a quick, autonomous response to prevent escalation.

Solution

Designed a compact mobile robot that navigates confined spaces, detects fire via thermal/IR sensors, and activates a suppression mechanism automatically.

Key Outcomes

  • Successfully built and tested working prototype.
  • Reduced response time in simulated fire tests.
  • Research paper peer-reviewed and published in IEEE.

Smart Garden Automated System

Overview

An IoT-based automation project designed to monitor environmental conditions and automate irrigation for plants to reduce water wastage and improve yield.

Hardware Used

  • ESP8266 Microcontroller
  • Soil Moisture Sensor & DHT11 (Temp/Humidity)
  • Submersible Water Pumps & Relays

Features

Real-time dashboard via Blynk, auto-shutoff for water conservation, and hardware LCD status display for local monitoring.

Weather App (Python + Tkinter)

Overview

A desktop application developed in Python that provides real-time weather information for any city using the OpenWeatherMap API.

Key Features

  • Fetches current temperature, humidity, and weather conditions.
  • Clean and user-friendly GUI built with Tkinter.
  • Error handling for invalid city names or network issues.
  • Visual icons representing weather states (Sunny, Cloudy, Rainy, etc.).

Development Process

Focused on API integration, JSON parsing, and creating a responsive GUI that works smoothly on different screen resolutions.
